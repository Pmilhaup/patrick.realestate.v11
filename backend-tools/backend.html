Component Details

    

      
🏘️ MLS Uploader
Daily MRED export → parse → filter 7 villages → upsert raw_sales → trigger snapshot.

      
📈 Infosparks Uploader
Monthly CSVs → metric detection → upsert historical_monthly_stats.

      
📊 Remine Uploader
3 XLSX files (Remine export caps) → upsert remine_property_index on PIN.

      
🧠 MilVal Algorithm
8.9% MAE predictability model. Scores 6,335 properties for listing likelihood.

      
📷 Daily Snapshots
compute_daily_snapshot() RPC aggregates raw_sales into daily_snapshots per village.

      
🔄 Automations
Zapier triggers + HubSpot CRM sync for prospect workflows.
